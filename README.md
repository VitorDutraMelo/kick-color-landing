# 🎨 Kick & Color — Football Coloring Books Landing Page

A modern, responsive landing page built to showcase and sell football-themed coloring books for kids.

Designed with a fun, playful UI and focused on conversion, this project highlights product features, benefits, and visual previews in a clean and engaging way.
